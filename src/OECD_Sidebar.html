About OECD Data

The Organisation for Economic Co-operation and Development (OECD) publishes over 1,400 statistical datasets covering macroeconomic indicators, labour markets, trade, prices, productivity, education, environment, and more. Data covers 38 member countries and many partner economies, updated regularly.

This add-on uses the OECD SDMX REST API to query data directly into your spreadsheet. No API key is required. Rate limit: 60 requests per hour.

Key Datasets

NAAG — National Accounts at a Glance (GDP) EO — Economic Outlook CPI — Consumer Price Indices LFS — Monthly Unemployment Rates BOP — Balance of Payments CLI — Composite Leading Indicators FINMARK — Financial Market Indicators KEI — Key Short-term Economic Indicators PDB — Productivity Database IDD — Income Distribution Database

Country Codes

OECD uses ISO 3166-1 alpha-3 country codes (e.g., USA, DEU, GBR, JPN, FRA). In the Explorer, available codes are loaded automatically for each dimension.

SDMX Format

Data is retrieved via the OECD SDMX REST API in JSON format. Time periods use SDMX conventions (e.g., “2024” for annual, “2024-Q1” for quarterly, “2024-06” for monthly).
