Analytics Functions
=BIS_GROWTH_RATE(range, periods)
Period-over-period growth rate (%). Default periods = 1.
Example: =BIS_GROWTH_RATE(B2:B100, 1)
=BIS_MOVING_AVG(range, window)
Simple moving average over a rolling window.
Example: =BIS_MOVING_AVG(B2:B100, 4) (4-quarter MA)
=BIS_YOY_CHANGE(range, periodsPerYear)
Year-over-year change (%). Set periodsPerYear to 12 (monthly), 4 (quarterly), or 1 (annual).
Example: =BIS_YOY_CHANGE(B2:B100, 12)
=BIS_SUMMARY(range)
Summary statistics: Count, Min, Max, Mean, Median, Std Dev.
Example: =BIS_SUMMARY(B2:B100)
Tips
  • Use the Explorer tab for full control over dataset selection and dimension filtering.
  • Use Quick Fetch for common one-click queries.
  • In dimension filters, hold Ctrl/Cmd to select multiple values.
  • Leave a dimension set to "(All)" to fetch all values for that dimension.
  • Use the analytics functions on fetched data columns for quick analysis.
  • The Start/End period fields accept formats like 2020, 2020-Q1, 2020-01.
