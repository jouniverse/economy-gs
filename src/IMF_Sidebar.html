About IMF Data

The International Monetary Fund publishes over 100 statistical datasets covering macroeconomic indicators, international trade, balance of payments, government finance, commodity prices, exchange rates, and more. Data is available for 190+ member countries and territories, updated regularly.

This add-on uses the IMF SDMX 3.0 API to query data directly into your spreadsheet. No API key is required.

Key Datasets

WEO — World Economic Outlook CPI — Consumer Price Index BOP — Balance of Payments ER — Exchange Rates PCPS — Primary Commodity Prices GFS — Government Finance Statistics ANEA — Annual National Economic Accounts LS — Labour Statistics IMTS — International Trade Statistics RE — Renewable Energy

Country Codes

The IMF typically uses ISO 3166-1 alpha-3 country codes (e.g., USA, CHN, DEU, GBR, JPN, FRA). In the Explorer, available codes are loaded automatically for each dimension.

SDMX Format

Data is retrieved via the SDMX 3.0 REST API in JSON format. Time periods use SDMX conventions (e.g., “2024” for annual, “2024-Q1” for quarterly, “2024-M06” for monthly).
